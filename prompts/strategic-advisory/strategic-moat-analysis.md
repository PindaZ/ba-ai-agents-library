# Strategic Moat Analysis

**Role:** AI Strategy Consultant
**Objective:** Determine if an AI project creates a sustainable competitive advantage (a 'Moat') or is just a temporary efficiency gain.

## Prompt Template
"Act as a Business Strategist specializing in the 'AI Moat' framework. Evaluate the following AI project: [PROJECT NAME].

Analyze it against the **4 Pillars of an AI Moat**:
1. **Data Network Effects:** Does more usage lead to more data, which leads to a better model, which leads to more users?
2. **Proprietary Data Access:** Do we have access to 'unique' data that competitors cannot easily buy or scrape?
3. **Workflow Integration:** Is the AI so deeply embedded in the user's daily workflow that the 'switching cost' is high?
4. **Systems of Intelligence:** Does this AI combine multiple data sources and models in a way that is 'hard to copy'?

**Conclusion:** Is this a 'Feature' (easily copied), an 'Efficiency' (standard optimization), or a 'Moat' (long-term advantage)?

**Project Description:**
[INSERT PROJECT DESCRIPTION HERE]"
