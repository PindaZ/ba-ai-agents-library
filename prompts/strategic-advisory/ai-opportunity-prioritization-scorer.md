# AI Opportunity Prioritization Scorer

**Role:** AI Business Analyst
**Objective:** Provide a rigorous, data-driven framework for scoring and ranking multiple AI use case ideas.

## Prompt Template
"Act as an AI Strategy Consultant. I will provide a list of potential AI use cases. Evaluate each one using the following weighted criteria (Scale 1-10):

1. **Business Value (40%):** Revenue growth, cost savings, or risk mitigation potential.
2. **Technical Feasibility (30%):** Data availability, model complexity, and integration ease.
3. **Organizational Readiness (20%):** Stakeholder buy-in and process maturity.
4. **Time to Value (10%):** How quickly a Pilot/MVP can be delivered.

**Format your response as a Markdown table:**
| Use Case | Value | Feasibility | Readiness | Speed | Total Score | Recommendation |
|----------|-------|-------------|-----------|-------|-------------|----------------|

**List of Use Cases:**
[INSERT USE CASES HERE]"
