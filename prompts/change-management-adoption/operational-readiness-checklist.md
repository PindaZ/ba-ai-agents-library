# Operational Readiness Checklist

**Role:** AI Business Analyst / Project Manager
**Objective:** A final, comprehensive 'Go/No-Go' checklist before deploying an AI system to production.

## Prompt Template
"Act as an AI Release Manager. We are 2 weeks away from the 'Go-Live' for [PROJECT NAME]. 

**Create a 'Launch Readiness Checklist' including:**
1. **Technical Readiness:** (e.g., 'Model latency is <X ms,' 'Rate limits are set,' 'Monitoring is active').
2. **Compliance Readiness:** (e.g., 'Data privacy audit is complete,' 'Model card is drafted').
3. **Support Readiness:** (e.g., 'Support team is trained,' 'FAQ is live,' 'Internal help docs are ready').
4. **Operational Readiness:** (e.g., 'Onboarding plan is active,' 'Stakeholder sign-off is documented').
5. **Rollback Strategy:** (e.g., 'What is the immediate action if the system fails in the first hour?').

**Conclusion:** Provide a 'Go/No-Go' Scorecard (1-10) for each category.

**Project Context:**
[INSERT PROJECT CONTEXT HERE]"
