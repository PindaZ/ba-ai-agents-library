# Onboarding Playbook

**Role:** AI Change Management Specialist / BA
**Objective:** Design a structured onboarding experience for new users of an AI system [AGENT NAME] to ensure high adoption.

## Prompt Template
"Act as a Customer Success Architect. I need a 'User Onboarding Playbook' for [AI SYSTEM NAME]. 

**The playbook must include:**
1. **Day 1: The 'Aha!' Moment:** What is the first thing a user should do to see the value of the AI immediately?
2. **First Week: Habit Building:** Suggest 3-5 tasks the user should try in their first week to integrate the AI into their workflow.
3. **The 'Prompt Guide':** Provide a list of 5 starter prompts for the user to try.
4. **Error Handling:** How should the user react when the AI makes a mistake?
5. **Support Channels:** Where should they go for help or to report issues?

**Conclusion:** Provide a 'New User Checklist' and a 3-part 'Welcome Email' series template.

**AI System Purpose:**
[INSERT SYSTEM PURPOSE HERE]"
