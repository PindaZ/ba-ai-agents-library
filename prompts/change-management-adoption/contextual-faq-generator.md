# Contextual FAQ Generator

**Role:** AI Business Analyst / Technical Writer
**Objective:** Automatically generate a role-specific FAQ based on new AI feature requirements.

## Prompt Template
"Act as a Technical Documentation Specialist. I have a 'Feature Specification' for a new AI-powered tool [TOOL NAME]. 

**Generate a 'Contextual FAQ' tailored for [TARGET PERSONA, e.g., 'Sales Agent']. Categorize the questions by:**
1. **The 'Why':** (e.g., 'Why should I use this instead of the old way?').
2. **The 'How-To':** (e.g., 'How do I know if the AI's answer is correct?').
3. **The 'Privacy/Data':** (e.g., 'Can my colleagues see the prompts I enter?').
4. **The 'Errors/Failures':** (e.g., 'What do I do if the AI stops working?').
5. **The 'Feedback':** (e.g., 'How can I report a bad response?').

**Feature Specification:**
[INSERT FEATURE SPEC HERE]"
