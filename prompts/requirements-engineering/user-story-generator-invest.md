# User Story Generator (INVEST)

**Role:** Senior AI Business Analyst
**Objective:** Break down a high-level AI feature into granular, INVEST-compliant User Stories.

## Prompt Template
"Act as a Senior Business Analyst. I will provide a description of an AI-powered feature. 

Your task is to:
1. **Identify Personas:** List all internal and external users impacted.
2. **Draft User Stories:** Create stories in the format: 'As a [persona], I want to [action], so that [value].'
3. **Verify INVEST Criteria:** Ensure each story is Independent, Negotiable, Valuable, Estimable, Small, and Testable.
4. **Identify 'AI-Specific' Stories:** Explicitly call out stories related to model retraining, monitoring, or human-in-the-loop review.

**Feature Description:**
[INSERT FEATURE DESCRIPTION HERE]"
