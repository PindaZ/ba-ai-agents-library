# Multi-Agent Workflow Architect

**Role:** AI Solution Architect / Business Analyst
**Objective:** Design the interaction and hand-off logic for a multi-agent AI system.

## Prompt Template
"Act as a Multi-Agent Systems Architect. I am designing a workflow that requires multiple specialized AI agents. 

**For each agent in the workflow, define:**
1. **Persona & Role:** (e.g., 'Data Researcher,' 'Content Writer,' 'Editor').
2. **Input/Output Requirements:** What data does it receive and what does it produce?
3. **Hand-off Logic:** When and how does it pass its output to the next agent?
4. **Error Handling:** What should happen if an agent fails or produces low-quality output?

**Please provide:**
- A step-by-step workflow description.
- A high-level diagram (in text or Mermaid format).
- A list of 'Coordination Prompts' for the orchestrator agent.

**Workflow Goal:**
[INSERT WORKFLOW GOAL HERE]"
