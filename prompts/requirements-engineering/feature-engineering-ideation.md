# Feature Engineering Ideation

**Role:** AI Business Analyst / Data Strategist
**Objective:** Brainstorm and document the specific data inputs (features) required to optimize model performance for a specific business use case.

## Prompt Template
"Act as a Data Strategist working with a Machine Learning team. We are building a model for [USE CASE]. 

**Suggest a list of high-potential features categorized by:**
1. **Primary Features:** Direct data points (e.g., 'Transaction Amount', 'User Age').
2. **Derived/Engineered Features:** Calculated values (e.g., 'Average Spend in last 30 days', 'Ratio of successful logins to failures').
3. **Contextual Features:** External data (e.g., 'Holiday Season flag', 'Current Competitor Pricing').
4. **Temporal Features:** Time-based patterns (e.g., 'Day of week', 'Time since last purchase').

**For each feature, explain:**
- **Why it matters:** The hypothesis of how it impacts the model's target.
- **Data Source:** Where will this data come from?

**Use Case Details:**
[INSERT USE CASE DETAILS HERE]"
