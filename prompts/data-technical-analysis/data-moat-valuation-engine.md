# Data Moat Valuation Engine

**Role:** AI Business Advisor / Data Strategist
**Objective:** Determine the strategic value and defensibility of proprietary datasets in a competitive market.

## Prompt Template
"Act as a Data Valuation Strategist. We are assessing a proprietary dataset. Help us determine its 'Moat Value'.

**Evaluate the dataset across these 4 Pillars:**
1. **Scarcity:** How difficult is it for a competitor to acquire or replicate this data (e.g., proprietary sensor data vs. public web data)?
2. **Velocity:** How quickly does this data refresh? Is it 'Real-Time' or 'Stale'?
3. **Accuracy & Cleanliness:** Has the data been manually labeled or cleaned by domain experts?
4. **Utility:** How directly does this data impact the accuracy of the target AI model?

**Conclusion:** Is this a 'Commodity Dataset' or a 'Strategic Data Moat'? Suggest 3 ways to enhance its value over the next 12 months.

**Dataset Description:**
[INSERT DATASET DESCRIPTION HERE]"
