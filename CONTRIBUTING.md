# Contributing to the BA & AI Agents Library

We welcome contributions to this library! To ensure consistency, please follow these guidelines when adding new prompts or skills.

## Adding a New Prompt

1. **Category:** Place the prompt in the correct functional toolkit directory.
2. **File Name:** Use lowercase with hyphens (e.g., `my-new-prompt.md`).
3. **Template Structure:**
   - **Role:** Define a clear persona (e.g., Senior BA).
   - **Objective:** Briefly state the goal.
   - **Prompt Template:** Use the standard Markdown formatting. Use `[PLACEHOLDERS]` in square brackets.

## Adding a New Skill

1. **Category:** Place the skill in the correct `skills/` subdirectory.
2. **File Name:** Use lowercase with hyphens (e.g., `my-new-skill.md`).
3. **Template Structure:**
   - **Category:** The toolkit it belongs to.
   - **Goal:** Clear objective.
   - **Prerequisites:** Information needed to start.
   - **Required Tools:** List any tools (e.g., Python, SQL, Web Search).
   - **Execution Workflow:** A step-by-step logic.
   - **Expected Output:** What the final result should be.

## Pull Request Process

1. Create a branch for your addition.
2. Ensure your prompt/skill follows the templates above.
3. Submit a Pull Request with a clear description of the new content.
