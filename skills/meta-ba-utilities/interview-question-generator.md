# Interview Question Generator Skill

**Category:** Meta-BA Utilities
**Goal:** Automatically generate targeted, persona-specific questions to elicit requirements during stakeholder interviews.

## Prerequisites
- Interviewee's job title.
- High-level project goal.

## Required Tools/Integrations
- `llm_prompt_generator`

## Execution Workflow
1. **Context Setup:** Define the project scope (e.g., "Automating invoice processing").
2. **Persona Mapping:** Analyze the interviewee's role (e.g., "Accounts Payable Clerk").
3. **Current State Elicitation:** Draft 3 questions to uncover their current manual pain points.
4. **Future State Elicitation:** Draft 3 questions asking what their ideal "magic wand" solution looks like.
5. **Constraint Elicitation:** Draft 2 questions to discover regulatory or technical blockers they deal with.

## Expected Output
A structured interview script tailored to extract maximum value from a 30-minute stakeholder meeting.
