# Training Module Creator Skill

**Category:** Change Management & Adoption
**Goal:** Automatically generate the syllabus and script for a training module on how to use a new AI feature.

## Prerequisites
- Feature specification or user manual.
- Target audience skill level (Beginner, Intermediate, Advanced).

## Required Tools/Integrations
- `text_summarizer`
- `presentation_script_generator`

## Execution Workflow
1. **Content Structuring:** Break the feature down into a 3-part lesson (Introduction, Live Demo, Hands-on Exercise).
2. **Concept Translation:** Translate technical AI terms (e.g., "Vector Search") into accessible analogies (e.g., "Like a highly organized library").
3. **Demo Scripting:** Write a step-by-step script for the trainer to follow during the live demonstration.
4. **Exercise Generation:** Create a safe, sandbox exercise for the user to try themselves.
5. **Knowledge Check:** Generate 3 multiple-choice questions to test understanding.

## Expected Output
A complete "Trainer's Guide" including presentation talking points, demo steps, and assessment questions.
