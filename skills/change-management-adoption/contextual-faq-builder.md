# Contextual FAQ Builder Skill

**Category:** Change Management & Adoption
**Goal:** Generate a role-specific FAQ document addressing the most likely questions users will have about the AI.

## Prerequisites
- Empathy map (to understand fears).
- System constraints document.

## Required Tools/Integrations
- `llm_faq_generator`

## Execution Workflow
1. **Question Brainstorming:** Generate 10 questions a specific user persona might ask.
2. **Categorization:** Sort questions into "How-to", "Data Privacy", "Troubleshooting", and "General".
3. **Drafting Answers:** Write clear, non-technical answers based on the system constraints.
4. **Tone Check:** Ensure the tone is reassuring and encouraging.
5. **Format Export:** Output as an HTML or Markdown accordion structure for easy embedding in an intranet.

## Expected Output
A ready-to-publish FAQ document specifically tailored to the concerns of the target user group.
