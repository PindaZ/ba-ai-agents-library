# Moat Analyzer Skill

**Category:** Strategic Advisory
**Goal:** Evaluate an AI initiative to determine if it builds a sustainable competitive advantage (a moat).

## Prerequisites
- Feature description.
- List of direct competitors.

## Required Tools/Integrations
- `google_web_search` (to check competitor features)

## Execution Workflow
1. **Data Network Effect Check:** Assess if the feature naturally collects proprietary data that improves the model over time.
2. **Competitor Scan:** Search the web to see if competitors already offer this exact feature via off-the-shelf APIs.
3. **Integration Depth:** Evaluate how deeply the AI feature is embedded into the user's daily workflow (switching cost).
4. **Scoring:** Assign a "Moat Score" (1-10) based on defensibility.
5. **Recommendation:** Suggest ways to increase defensibility (e.g., locking in exclusive data partnerships).

## Expected Output
A "Moat Defense Profile" detailing the feature's vulnerability to replication and strategies to strengthen it.
