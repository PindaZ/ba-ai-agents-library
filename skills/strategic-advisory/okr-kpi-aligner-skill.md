# OKR/KPI Aligner Skill

**Category:** Strategic Advisory
**Goal:** Ensure that an AI project's technical metrics are directly tied to high-level business OKRs.

## Prerequisites
- Corporate OKRs (Objectives and Key Results).
- Technical metrics of the AI model (e.g., F1 score, latency).

## Required Tools/Integrations
- `read_file`

## Execution Workflow
1. **Objective Ingestion:** Read the corporate Objective (e.g., "Increase Customer Retention").
2. **Key Result Ingestion:** Read the specific target (e.g., "Reduce churn by 5%").
3. **Mapping:** Connect the AI model's output (e.g., "Churn prediction accuracy") to the Key Result.
4. **Leading Indicator Definition:** Define how the technical metric acts as a leading indicator for the business metric.
5. **Dashboard Spec:** Draft the requirements for a dashboard that displays both metrics side-by-side.

## Expected Output
An "Alignment Matrix" that maps Model Metrics -> Operational KPIs -> Strategic OKRs.
