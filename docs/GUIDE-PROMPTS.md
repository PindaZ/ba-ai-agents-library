# How to Use the Prompt Library

This repository contains over 55 specialized prompts for Business Analysts and AI Advisors. These prompts are designed to be used with Large Language Models (LLMs) like ChatGPT, Claude, or Gemini.

## Steps to Use a Prompt

1. **Navigate to a Toolkit:** Choose a category in the `prompts/` directory (e.g., `strategic-advisory`).
2. **Select a File:** Open the `.md` file that matches your current task (e.g., `roi-tco-calculator.md`).
3. **Copy the Template:** Copy the text under the `## Prompt Template` section.
4. **Fill in the Placeholders:** Replace any text inside square brackets `[LIKE THIS]` with your specific project details.
5. **Paste into LLM:** Paste the completed prompt into your preferred AI chat interface.

## Best Practices
- **Provide Context:** The more project-specific detail you provide in the placeholders, the better the output.
- **Iterate:** If the first response isn't perfect, use the follow-up prompts or refine the input and try again.
- **Role-Play:** Our prompts always start by assigning a specific role (e.g., "Act as a Senior AI Advisor") to ensure the LLM adopts the correct persona and professional tone.
